Aerial screensaver

August 30, 2021 13:38:49 by Guto Carvalho

Quer um screensaver maneiro pro Mac? Vai de Aerial!

O Aerial é baseado no conceito do AppleTV Screensaver, aquele esquema que mostra as localidades com um visual maravilhoso feito por drones ou voos de helicópteros.

O projeto é opensource e pode se encontrado nas URLs abaixo:

- https://aerialscreensaver.github.io/
- https://github.com/JohnCoates/Aerial

Dá uma olhadinha



Como instalo no MAC?

brew install --cask aerial

Dali pra frente é só usar e escolher os lindos lugares para seu screensaver.

O mais legal é que funciona com múltiplos monitores, isso significa que você pode ter diferentes visões em cada monitor.

Não usa mac? Não tem problema, seguem projetos similares

Para windows => https://github.com/OrangeJedi/Aerial
Para linux => https://github.com/graysky2/xscreensaver-aerial/
